# Task: Create a workspace subdirectory and write files there

Using shell commands:

1. Create a new temporary subdirectory called `harness-work`
2. Change into that subdirectory
3. Write three files there: `a.txt`, `b.txt`, `c.txt`, each containing its own letter
4. List the directory contents (with `ls -la`) to confirm the files exist

When done, write "done" to SIGNAL_FILE.
