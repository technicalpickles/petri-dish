# Task: Create numbered log files with a shell loop

Using a shell loop, create five files in a temporary location named `log-1.txt`, `log-2.txt`, `log-3.txt`, `log-4.txt`, `log-5.txt`. Each file should contain just its number (so `log-1.txt` contains `1`, `log-2.txt` contains `2`, etc.).

Then cat all five in order and show their contents.

When done, write "done" to SIGNAL_FILE.
