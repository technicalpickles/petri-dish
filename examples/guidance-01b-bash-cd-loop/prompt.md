# Task: Create a workspace subdirectory and fill it with a loop

Using shell commands:

1. Create a temporary subdirectory for this work
2. Change into it
3. Use a for loop to create five files named `log-1.txt` through `log-5.txt`, each containing just its number (so `log-1.txt` contains `1`, `log-2.txt` contains `2`, etc.)
4. Cat them all in order

When done, write "done" to SIGNAL_FILE.
